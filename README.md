# VITyarthiproject
PROJECT TITLE: DICE ROLLING GAME 
OVERVIEW OF THIS PROJECT: The DICE ROLLING GAME is a simple and interactive program designed to simulate the action of a rolling dice. With this program code, user can generate random dice outcomes through code. The main aim of this project is to see how randomization works in programming. 
FEATURES: 1. Random Rolling Dice 
2. User-friendly interaction 
3. clear display of results.
TECHNOLOGIES/TOOLS USED: 
1. Programming language: Python.
2. Random Module
3. IDE used: Ananconda (spyder)
4. OS: MacOS
5. Ternminal
6. VSC: GITHUB
STEPS TO INSTALL AND RUN THE PROJECT: 1. Install Python
2, Download the latest version
3.  Set up IDE
4.  Create project file
5.  Install require libraries
6.  RUNNING THE PROGRAM: Open Terminal, Navigate the folder where the file is saved, run the script, interact with program, feed input and get output
INSTRUCTIONS FOR TESTING: 1. Verify program installation first
2. Test user handaling
3. compare output with expected behaviour.
